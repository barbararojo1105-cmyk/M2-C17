# Proyecto Git y GitHub

Este proyecto corresponde a la actividad número 7 de la clase 17 del módulo 2, del curso de Front-end; en el cual el objetivo es aprender a trabajar con Git y GitHub.

##  Descripción

Se creó un repositorio local que luego fue vinculado a un repositorio remoto en GitHub. Se trabajó con ramas para simular el desarrollo de nuevas funcionalidades y se realizaron fusiones (merge) mediante Pull Request.

##  Funcionalidades implementadas

- Creación de repositorio local con Git
- Conexión con repositorio remoto en GitHub
- Uso de ramas para nuevas funcionalidades.
- Edición del archivo index.html
- Uso de commits para guardar cambios

##  Ramas utilizadas

- main → rama principal del proyecto
- feature/navbar → implementación de barra de navegación
- feature/footer → implementación de footer
